# Media Feature

## Components
- `FolderList`: Displays a grid of folders.
- `FileList`: Displays a list of files.
- `Breadcrumbs`: Navigation path.
- `CreateFolderModal`: Modal to create new folders.
- `UploadModal`: Modal to upload files.

## Pages
- `MediaPage`: Main media browser view.
- `FileDetailsPage`: Detailed view of a file with transcription/analysis tools.

## Services
- `mediaService`: Handles API calls for folders and files.
